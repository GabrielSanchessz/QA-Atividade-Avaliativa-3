Atividade Avaliativa 3 - Testes E2E 
Objetivo 
Desenvolver (em grupo com 3 ou 4 pessoas) testes ponta a ponta de uma das 
aplicações dos seguintes repositórios (github.com/juice-shop/juice-shop ou 
github.com/guilherme-ferraz/qa-atividade-avaliativa-2), usando uma das seguintes 
ferramentas (playwrite, selenium ou cypress), cobrindo todas as usabilidades esperadas de 
um usuário comum do sistema (listagem, inserção, edição e exclusão de registros). 
Instruções - - - - 
Clone o repositório escolhido e execute a aplicação localmente 
Crie um repositório próprio de execução dos testes 
Compartilhe o repositório com do código de testes com o usuário `guilherme-ferraz` 
Use um ambiente com container(s) Docker para instalação das dependências do 
projeto. 
Data de Entrega: 30/06 
Entregáveis: - - 
Compartilhamento do repositório com o usuário `guilherme-ferraz` 
Apresentação no laboratório da execução dos testes (demonstração em tempo real, 
descrição em slides e/ou gravação) dividido em 3 etapas (1 - acesso/listagem das 
páginas e inclusão de novos registros; 2 - edição de registros existentes; 3 - 
exclusão dos registros inseridos no teste)
